Mini Paginator by Wilson Fernandez (English instructions)

"Mini Paginator", made entirely in php, designed in the simplest possible way for implementation, the script consists of a single php file that contains the "paginator" class that gets the parameters from a MySQL query and calculate, based on a maximum number of results per page, how many pages are total displayed, together with the class file its one css file that contains 2 different styles, one light and another one dark, the class can use this css, or another one choosen by the user.

The data required to run the class are three

$total_results = $num_rows; // total results from a query
$maximum_per_page = 3; // maximum results to show per page
$maximun_links = 5; // maximum links to show

Implementation

First copy the folder "Paginator" to the directory where the pagination is required, then use the following sample code.

The most important

The first query gives the total data to be displayed and store it in $total_results.

Then we define the parameters that we want, how many results per page we want to $maximun_per_page and maximum of links to display in $maximun_links.

Then we come to the process of paging, here we choose the style , "Light" or "Dark".

We continue the process by the second query with the "LIMIT" clausule where we use the parameter $page->limit, here is where the magic is made.

The last thing is show the pagination results.

The demo of this code can see it in http://multicoders.com/scripts/mini_paginator/test.php

Mini Paginator by Wilson Fernandez (Instrucciones en español)

"Mini Paginator" es un script hecho íntegramente en php, pensado en la forma de implementación más simple posible, el script consta de un solo archivo php que contiene la clase "paginator" que recibe los parámetros de una consulta MySQL y calcula, en base a un numero de máximos resultados por página, en cuantas páginas se mostrara el total de resultados, adicionalmente junto con el archivo de la clase se encuentra un archivo css que contiene 2 estilos diferentes, uno claro y otro oscuro, la clase puede usar este css u otro que el usuario elija.

Los datos que requiere la clase para poder funcionar son 3

$total_results = $num_rows; // total de resultados a paginar
$maximum_per_page = 3; // máximo de resultados por pagina
$maximun_links = 5; // máximo de links a mostrar

Modo de implementación

Primero copiar la carpeta “paginator” al directorio donde se requiere la paginación, luego seguir el siguiente código de ejemplo.

Lo más importante

En el primer query obtenemos el total de datos a mostrar y lo guardamos en $total_results.

Luego definimos los parámetros que queremos a nuestro antojo, cuantos resultados por página queremos en $maximun_per_page y el máximo de links a mostrar en $maximun_links.

Luego llegamos al proceso de paginación, aquí elegimos el estilo del paginador, “Light” o “Dark”.

Continuamos con el proceso haciendo la segunda query con la cláusula “LIMIT” donde usamos el parámetro $page->limit, aquí es donde se hace la magia.

Lo último que nos falta es mostrar el resultado de la paginación.

El demo de este código puedes verlo en http://multicoders.com/scripts/mini_paginator/test.php
